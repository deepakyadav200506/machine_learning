### DIABETES PREDICTION USING MACHINE LEARNING 
### Overview
his project predicts whether a person is Diabetic or Not based on 
medical input parameters using a Machine Learning classification model.
<br>
this project is part of a flipped course evaluation 
and demonstrates how to:
- Load and preprocess data
- Train and evaluate a regression model
- Visualize the relationship between features and target
- Use Git and GitHub for version control
<br>
### Features
 - Pregnancies  
 - Glucose 
 - BloodPressure 
 - SkinThickness 
 - Insulin 
 - BMI 
 - DiabetesPedigreeFunction 
 - Age 
 - predict whether the person is diabetic or not

   ### Technologies/Tools Used
   - Python
   - Numpy
   - Pandas
   - StandardScaler
   - Git & Github
