### PROBLEM STATEMENT
diabetes is one of the most common chronic diseases in the world.
According to the World Health Organization (WHO), over 422 million 
people worldwide are affected by diabetes. Early detection of diabetes 
can help patients take preventive measures and get timely treatment, 
which can significantly reduce health complications.
<br>
### OBJECTIVE
The objective of this project is to build a Machine Learning model 
that can predict whether a patient is diabetic or not, based on 
diagnostic measurements such as glucose level, blood pressure, 
BMI, age, and other medical features.
<br>
### PROPOSED SOLUTION  
A supervised Machine Learning classification model is trained on 
the Pima Indians Diabetes Dataset. The model takes 8 medical input 
parameters and predicts whether the person is diabetic or not.
<br>
### STEPS FOLLOWED
1. Data Collection
2. Data Pre-processing and Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Scaling using StandardScaler
5. Model Training
6. Model Evaluation
7. Prediction on new input data
  ### EXPECTED OUTCOME
  The system will output one of the following results based on 
the input data:
-  "The person is NOT Diabetic"
- "The person IS Diabetic"
